# Project Documentation — Global Superstore Analysis

## Project Background
The Global Superstore dataset was analyzed to understand how sales performance translates into profitability across regions, product categories, and customer segments.

The project focuses on identifying:
- profit leakages
- operational inefficiencies
- high-performing areas
- data-driven opportunities for business improvement

---

## Business Questions

The analysis was guided by the following key questions:

1. Is the business profitable overall?
2. Which regions drive the most profit?
3. Which products generate losses?
4. How do discounts affect profitability?
5. Do product returns reduce profit?
6. Which customer segments are most valuable?

---

## Methodology

The project followed a structured analytics workflow:

### Stage 1 — Excel
- Initial inspection
- Data cleaning
- Structure validation

### Stage 2 — SQL
- Table creation
- Data import
- Data transformation
- Aggregation and analysis queries

### Stage 3 — Power BI
- KPI creation
- Dashboard development
- Data modeling
- Visualization and storytelling

---

## Analytical Approach

The analysis focused on:
- revenue vs profitability
- operational inefficiencies
- product performance
- regional performance
- discount impact
- returns impact

---

## Assumptions
- Dataset represents real business operations
- Sales and profit values are accurate
- Returns are correctly recorded

---

## Limitations
